What is a Flex Box?

Hello, this is Tanaka from Progate My name is Tanaka from Path. In this article, I'd like to introduce a more advanced use of flexbox. CSS Flexbox (Flexible Box LayoutModule) is a layout model for efficiently arranging, aligning, and distributing items within a container. Flex boxes are controlled by properties applied to the parent element (flex container) and child elements (flex items). This allows items of variable width and height to be easily adjusted to achieve responsive design for a variety of devices and screen sizes.

display: flex;

To use a flexbox, first add a parent element with display:. flex; or display: inline-flex; to the parent element. inline-flex; to the parent element. This makes the element a flex container and the child elements flex items.
